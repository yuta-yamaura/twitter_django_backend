## 起動方法

以下のディレクトリに移動

```
$ cd twitter_django_backend
```

Django の仮装環境を作成

```
$ python -m venv venv
```

Django の仮装環境を起動 ※venv ディレクトリと同じ階層で実行

```
$ source venv/bin/activate
```

以下のディレクトリに移動

```
$ cd backend/
$ pwd で以下のディレクトリになってればOK
/Users/user/Projects/Xclone/backend
```

コンテナを起動

```
$ docker-compose up
```
